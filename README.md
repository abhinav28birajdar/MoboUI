# Mobile UI Kit

A production-ready Mobile UI Component Library Platform for Flutter, React Native, and Expo.

## Features

- 📱 **Component Library**: 300+ components with live previews.
- 🎨 **Playground**: Real-time code editing with Monaco Editor.
- 🚀 **Expo & Flutter Support**: Live emulation via Expo Snack and DartPad.
- 🌗 **Dark Mode**: Fully supported.
- ♿ **Accessible**: ARIA compliant components.
- ⚡ **Performance**: Optimized for speed and SEO.

## Tech Stack

- **Frontend**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **UI**: shadcn/ui, Framer Motion, Lucide React
- **State**: Zustand
- **Editor**: Monaco Editor

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/`: Next.js App Router pages
- `components/`: React components (UI, Layout, specialized)
- `lib/`: Utilities, data, stores
- `types/`: TypeScript definitions
