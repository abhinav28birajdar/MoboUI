# Getting Started with MOBOUI

MOBOUI is a collection of premium, production-ready UI components for Flutter, React Native, and Expo. Our goal is to bridge the gap between design and development by providing high-quality, copy-pasteable code snippets that look amazing out of the box.

## Core Principles

- **Premium Design**: We prioritize aesthetics. No more boring default styles.
- **Framework Agnostic**: Every component is available for Flutter, RN, and Expo.
- **Copy & Paste**: No complex libraries to install. Just copy the code and own it.
- **Fully Customizable**: Use props to tweak everything to your brand.

## Quick Start

1. **Browse Components**: Head over to the [Catalog](/components) and find a component you like.
2. **Preview & Customize**: Use the [Playground](/playground) to test different props.
3. **Copy Code**: Select your framework (Flutter, RN, or Expo) and click copy.
4. **Ship**: Paste the code into your project and you're good to go!

## Prerequisites

Before using MOBOUI, ensure you have the following installed:

### Flutter
- Flutter SDK 3.x or higher
- Dart 3.x

### React Native / Expo
- Node.js 18+
- React Native 0.72+
- Expo 49+ (if using Expo)
